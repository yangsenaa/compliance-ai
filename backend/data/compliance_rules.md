# 催收合规质检规则

## 一、严禁行为（高严重度）

### 1.1 威胁恐吓
- 严禁使用"坐牢"、"拘留"、"起诉"、"逮捕"等法律威胁
- 严禁使用"上门"、"找你麻烦"等人身威胁
- 严禁以暴力或非法手段恐吓借款人
- 违规示例："你再不还款就让你坐牢"、"我们会派人上门找你"

### 1.2 骚扰第三方
- 严禁联系借款人的家属、同事、朋友（除担保人外）
- 严禁向第三方透露借款人的债务信息
- 严禁威胁联系借款人所在单位
- 违规示例："我要打电话给你们公司领导"、"我要告诉你家人你欠债的事"

### 1.3 侮辱性语言
- 严禁使用"废物"、"骗子"、"流氓"、"无赖"等侮辱词汇
- 严禁对借款人进行人身攻击或诽谤
- 严禁使用歧视性语言
- 违规示例："你这个骗子"、"你真是个废物"

### 1.4 虚假陈述
- 严禁谎称自己是执法人员或律师
- 严禁虚报欠款金额或利率
- 严禁以不存在的法律后果恐吓
- 违规示例："我是警察，你明天必须到派出所报到"

## 二、规范行为（中严重度）

### 2.1 联系时段
- 工作日 8:00-21:00 可以联系
- 节假日 9:00-20:00 可以联系
- 禁止在深夜（21:00后）联系
- 禁止在凌晨（8:00前）联系
- 违规示例：凌晨3点发送催收短信或打电话

### 2.2 话术规范
- 必须先自我介绍：公司名称 + 工号
- 必须明确说明通话目的（催收事由）
- 每次通话不得超过3分钟（特殊情况除外）
- 每日联系次数不得超过3次
- 违规示例：不报公司直接催款、每天打10次电话

### 2.3 还款协商
- 必须给借款人充分表达的机会
- 不得强迫借款人立即还款（需给予合理缓冲期）
- 应按照合同约定的利率计算欠款
- 违规示例："你现在必须立刻还款，否则后果自负"

### 2.4 信息核实
- 联系前需核实对方身份确为借款人本人
- 不得对非借款人施压
- 违规示例：误拨他人电话后仍继续催收

## 三、提示行为（低严重度）

### 3.1 记录要求
- 应记录每次通话的时间、内容和结果
- 应在系统中及时更新借款人还款状态
- 应保存通话录音（按规定期限）
- 提示：未及时记录可能导致合规风险

### 3.2 还款安排
- 应主动询问借款人还款困难的原因
- 应提供分期还款方案
- 应介绍可用的还款渠道
- 提示：主动提供解决方案有助于提高回款率

### 3.3 投诉告知
- 应告知借款人投诉渠道（客服电话/邮箱）
- 应告知借款人有权要求停止联系
- 提示：主动告知权利有助于维护良好的客户关系

### 3.4 语言礼貌
- 建议全程保持礼貌用语
- 建议使用"您"而非"你"称呼借款人
- 提示：礼貌沟通有助于提高借款人配合意愿

## 四、合规操作流程

### 4.1 标准开场白
```
"您好，我是[公司名称]的催收专员，工号[XXX]。
请问您是[借款人姓名]吗？
我联系您是关于您在[日期]的[金额]贷款还款事宜。"
```

### 4.2 还款困难处理流程
1. 倾听借款人说明困难原因
2. 评估还款能力，提供合理方案
3. 协商分期还款计划
4. 记录协商结果并跟进

### 4.3 拒绝还款处理流程
1. 说明逾期后果（仅限合法后果）
2. 告知信用影响
3. 提供最后期限
4. 按程序上报处理

## 五、法律法规依据

- 《中华人民共和国民法典》相关条款
- 《互联网金融个人网络借贷信息中介机构业务活动管理暂行办法》
- 中国人民银行关于规范贷款催收行为的相关规定
- 《消费者权益保护法》相关条款
